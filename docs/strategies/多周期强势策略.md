# 多周期强势策略（multi · v1.0）

> 版本：v1.0 · 2026-06  
> 状态：**已实现**  
> 对应小程序 Tab：`multi`（多周期强势）  
> 对应后端枚举：`StrategyTypeEnum.MULTI_STRENGTH`（code = `multi`）  
> 策略类：`MultiStrengthStrategy`

---

## 1. 产品定义

**一句话**：日/周/月结构同步偏强，且日 K 在波段或梯子关键位出现突破确认。

| 决策 | 结论 |
|------|------|
| 主路径 | **多周期共振（Context ≥ 2）+ 日 K 突破（Trigger ≥ 1）** |
| 弱环境 | B/C 档默认**不进池**（`tierMin=A`） |
| 列表 | 命中标的按 **score 降序 + 分页** |
| 与 qsn 边界 | 不用 MA 延续为主路径；主工具 Band / RevertBand / TiZi |

小程序文案：**日周月共振 · 波段/梯子突破 · 强势延续**

---

## 2. 总体判定

```
命中 = Gate  AND  Context 共振 ≥ minPeriods（默认 2/3）
     AND  Trigger ≥ 1（A/B/C 至少一种）
     AND  档位 ≥ tierMin（默认 A，即仅 S/A 进池）
排序 = score
```

| 档位 | 条件 | 默认进池 |
|------|------|----------|
| **S** | 3 周期 Context + 多信号/多周期突破 | ✅ |
| **A** | ≥2 周期 Context + Trigger | ✅ |
| **B** | ≥2 周期 Context、无 Trigger | ❌ |
| **C** | 1 周期 Context + Trigger | ❌ |

---

## 3. Gate

| # | 条件 |
|---|------|
| G1 | 在 `filterStockMap` 内 |
| G2 | 近 6 日平均成交额 > 5000 万 |
| G3 | 周或月未跌破近端支撑 Low |

---

## 4. Context（日/周/月）

每周期需同时：**趋势波段 Low 之上** 且 **反转波段 Low 之上**  
加分（计分）：收盘 > MA20、MACD > 0

---

## 5. Trigger（日 K 操作）

| 模式 | 工具 |
|------|------|
| A 趋势波段突破 | `SignalBandTools.checkCrossBandHighSignal` |
| B 反转波段突破 | `SignalBandTools.checkRevertBandHighSignal` |
| C 梯子 High 突破 | `SignalTools.checkCrossTiZiHighSignal` |

可在日/周/月 trendPeriod 上分别检测，用于 score 与 S 档。

---

## 6. 客户端参数（`m*`）

| 参数 | 默认 |
|------|------|
| `mMinResonancePeriods` | 2 |
| `mMinAmountWan` | 5000 |
| `mEnableModeA/B/C` | 1 |
| `mEnableWeakContext` | 0 |
| `mEnableWeakSingle` | 0 |
| `mTierMin` | A |
